## Краткое описание
Поздравлятор - консольное приложение для хранения и отображения списка дней рождения.
Основные возможности:
- Отображение сегодняшних, ближайших и прошедших (+ и - неделя соответственно) дней рождения
- Отображение общего списка дней рождения
- Автоматический подсчет возраста именинника на момент ДР для отображения в консоли
- Добавление, удаление и редактирование дней рождения

## Стек
- Java 17+
- JDBC
- java.time (LocalDate, MonthDay)
- java.util (Scanner, List, ArrayList, Properties)
- java.sql (Connection, PreparedStatement, ResultSet, Date)
- PostgreSQL

## Конфигурация
Для корректной работы программы необходимо:
- Создать БД в PostgreSQL. Таблица birthday создается автоматически при запуске программы
- Создать в корневой папке с программой файл config.properties и указать в нем следующие параметры:

db.url=jdbc:postgresql://<хост>:<порт>/<имя_БД>
db.user=<имя_пользователя> (по умолчанию postgres)
db.password=<пароль>

Пример файла config.properties:
```
db.url=jdbc:postgresql://localhost:5432/birthdays
db.user=postgres
db.password=admin
```
